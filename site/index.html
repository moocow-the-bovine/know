Boards

(<= 12KB)
Digital images of the game board for printing, in a variety of paper sizes. Requires a PDF interpreter such as Ghostscript or Adobe's Acrobat Reader.
  • ISO Formats (metric)
    • ISO A4
      Board Diameter = 185mm
      Page Size = 210mm x 297mm
    • ISO A3
      Board Diameter = 272mm
      Page Size = 297mm x 420mm
    • ISO A2
      Board Diameter = 395mm
      Page Size = 420mm x 594mm
    • ISO A1
      Board Diameter = 569mm
      Page Size = 594mm x 841mm
    • ISO A0
      Board Diameter = 816mm
      Page Size = 841mm x 1189mm
  • ANSI Formats (imperial)
    • ANSI-A (letter)
      Board Diameter = 7.5"
      Page Size = 8.5" x 11"
    • ANSI-B
      Board Diameter = 10"
      Page Size = 11" x 17"
    • ANSI-C
      Board Diameter = 16"
      Page Size = 17" x 22"
    • ANSI-D
      Board Diameter = 21"
      Page Size = 22" x 34"
    • ANSI-E
      Board Diameter = 33"
      Page Size = 34" x 44"
